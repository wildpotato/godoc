Download

Visit the official download page then click on the Download button.

  • Linux

    1. Extract the archive you downloaded into /usr/local, creating a Go tree in /usr/local/go. This step also removes any previous installation of Go at /usr/local/go. You may want to back up if needed. 

      $ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.16.2.linux-amd64.tar.gz

    2. Add /usr/local/go/bin to the PATH environment variable.

      $ export PATH=$PATH:/usr/local/go/bin

    3. Verify that you've installed Go by opening a command prompt and typing the following command:

      $ go version

    4. Confirm that the command prints the installed version of Go.
