Built-in Types

Go has only four built-in types

  1. Integers

    • + Addition

    • - Subtraction

    • * Multiplication

    • / Division

    • % Remaineder

  2. Floating-point numbers

    • float32 single precision, takes up 32 bits

    • float64 double precision, takes up 64 bits

    • complex64 for complex numbers, takes up 64 bits

    • complex128 for complex numbers, takes up 128 bits

  3. Strings

    • Use len to get size of string

    • Strings are zero-indexed, "Hello World"[1] returns 'e'.

    • Can be created by either double quotes "Hello" or backticks `Hello` with the difference being that double quotes can not contain any newlines but only escape characters.

  4. Booleans

    • && and

    • || or

    • ! not
